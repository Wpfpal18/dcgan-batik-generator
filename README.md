DCGAN Batik Generator

Proyek ini merupakan Final Project mata kuliah Deep Learning yang mengimplementasikan Deep Convolutional Generative Adversarial Network (DCGAN) untuk menghasilkan citra motif batik Indonesia secara sintetis. Model dilatih menggunakan dataset publik Indonesian Batik Motifs.

Training model dilakukan menggunakan TensorFlow/Keras, sementara hasil model diintegrasikan ke dalam aplikasi berbasis Streamlit agar pengguna dapat menghasilkan gambar batik secara interaktif. Fokus proyek ini adalah eksplorasi kemampuan model generatif dalam mempelajari pola visual batik.

Dataset tidak disertakan di repository karena keterbatasan ukuran dan dapat diunduh dari Kaggle:
https://www.kaggle.com/datasets/dionisiusdh/indonesian-batik-motifs

Dataset diletakkan pada folder project/data/.

Detail metodologi, eksperimen, evaluasi hasil, serta pembahasan lengkap tersedia pada laporan proyek (PDF) yang disertakan dalam repository.
